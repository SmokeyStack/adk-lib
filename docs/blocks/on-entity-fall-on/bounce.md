# adk_lib:on_entity_fall_on_bounce

## What does it do?

This component will make the entity that falls on this block to bounce.

## How to use

Add `"adk_lib:on_entity_fall_on_bounce": {}` to your block json file.

### Parameters

- `"force"`: The amount of force to apply. Default value is `1.0`.

### Example

```json
"adk_lib:on_entity_fall_on_bounce": {
    "force": 0.8
}
```
