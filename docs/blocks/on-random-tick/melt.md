# adk_lib:on_random_tick_melt

## What does it do?

This component will turn your block into the melted state when a player destroys the block.

## How to use

Add `"adk_lib:on_random_tick_melt": {}` to your block json file.

### Parameters

- `"melted_state"`: The melted state

### Example

```json
"adk_lib:on_random_tick_melt": {
    "melted_state": "minecraft:water"
}
```
