# adk_lib:on_place_turn_into

## What does it do?

This component allows the block to turn into another block when it is placed.

## How to use

Add `"adk_lib:on_place_turn_into": {}` to your block json file.

### Parameters

- `"block"`: Identifier of the block to turn into

### Example

```json
"adk_lib:on_place_turn_into": {
    "block": "minecraft:bedrock"
}
```
