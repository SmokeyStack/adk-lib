# adk_lib:on_random_tick_sugar_cane

## What does it do?

This component will make the block advance their age similar to vanilla sugar cane. This component requires you to have the block state, `{namespace}:age`, which is an int property.

## How to use

Add `adk_lib:on_random_tick_sugar_cane` to the `minecraft:custom_components` array in your item json file.
