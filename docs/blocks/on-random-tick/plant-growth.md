# adk_lib:on_random_tick_plant_growth

## What does it do?

This component will make the block advance their age similar to vanilla crops. This component requires you to have the block state, `adk_lib:age`, which is an int property.

## How to use

Add `adk_lib:on_random_tick_plant_growth` to the `minecraft:custom_components` array in your item json file.
